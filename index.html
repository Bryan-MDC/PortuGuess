Como jogar PortuGuess
🎭 Conceito

No início de cada partida, todos recebem uma função secreta. Os Inocentes conhecem a palavra e devem identificar os Impostores. Os Impostores não conhecem a palavra e precisam sobreviver sem ser descobertos. Os Coringas têm objetivos especiais e buscam ser eliminados.

⚔️ Classes
🟢 Inocente
🎯 Principal Eliminar todos os Impostores. (+1 pt)
⭐ Extra Não eliminar o Coringa durante a partida. (+1 pt)
🔴 Impostor
🎯 Principal Sobreviver até o final da partida. (+1 pt)
⭐ Extra Se sobreviver, usar a palavra corretamente em uma frase. (+1 pt)
🟡 Coringa
🎯 Principal Ser eliminado pelos outros jogadores. (+1 pt)
⭐ Extra Ser eliminado na primeira rodada. (+1 pt)

🗣️ Sistema de Falas
  • Cada jogador dá uma pista sobre a palavra sorteada.
  • É proibido revelar a palavra diretamente.
  • Os Impostores devem usar as pistas para descobrir a palavra sem se trair.
  • Partidas com 6 ou 8 jogadores: 1 ciclo de falas. Com 10: 1 ciclo.
🗳️ Votação
  • Após o ciclo de falas, todos votam simultaneamente.
  • Não é permitido votar em si mesmo.
  • O voto não pode ser alterado após confirmação.
  • É permitido votar em Ninguém.
  • Empate: ninguém é eliminado.
  • Maioria para Ninguém: novo ciclo de falas.
  • As rodadas continuam até restar metade dos jogadores iniciais.
🏆 Condições de Vitória
Inocentes Todos os Impostores são eliminados.
Impostores Pelo menos um Impostor permanece vivo ao final.
Coringa O Coringa é eliminado pelos demais participantes.
